在因pa2生存艰难之余对即将到来的硬核算法期中进行一些徒劳的复习。

作业回顾

ProblemSet 1

P1 循环不变式

  • 要想证明BubbleSort确实对原先的元素进行了排序，我们就还要证明经过排序的数组中的元素都是原数组中的元素。

  • 循环不变式性质：

    初始化：循环的第一次迭代前，它为真。

    保持：如果循环的每一次迭代前它为真，那么下次迭代前它仍为真。

    终止：在循环终止时，不变式为我们提供一个有用的性质，该性质有助于证明算法是正确的。

  • 如何寻找循环不变式：

    算法是一步一步执行的，如果每一步（包括初始和结束）都满足同一个条件，那么这个条件就是要找的循环不变式。

    每一层循环都有不同的循环不变式，要注意每一次循环改变了什么，达成了什么目的。

  • 循环不变式的证明：

    （basis）在第一次迭代前，不变式成立

    （inductive step）假设在某一次循环前成立的，那么在本次循环之后，性质仍然保持。

    （termination）程序可以终止而且可以得到想要的结果。

P2 计算时间复杂度
  • 简单的迭代的复杂度还是容易计算的。
  • 一个简单的循环不变式的证明。
P3/4 一些渐近的练习

  • 复习笔记后得到常用的关系（记得背）

    常量© <$loglogn$ < 对数$logn$ < $log^2n$ < $\sqrt n$<线性$n$ < $nlogn$ < $n^2$ < $n^3(n^c)$ <. $2^n$ < $n!$

    Ps: 若$c\geq 1$，大于线性；若$c\leq 1$，则小于线性。

  • 如果出现非常见的：

    • 定义法：根据渐近符号的定义

    • 极限法：$\lim _{n\rightarrow \infty}\frac{f(n)}{g(n)}$

      =0, 表明$f(n)\in O(g(n))$

      =c, 表明$f(n)\in \Theta(g(n))$

      =$\infty$，表明$f(n)\in \Omega(g(n))$

    • 取对数法（有用但是笨蛋hanny还没有掌握）：

      $log_a(MN)=log_aM + log_aN$

      得到一般的和的形式后即可舍去低阶项。

      • 针对复合乘积
      • 一次取对数后，还可以选择性使用整体替代法

  • 作业里出现的一个$e^n$，其<<$n!$

P5 用一个数组实现两个栈（共享栈）

  • 主要思路：

    维护两个指针$p_1$和$p_2$，分别指向两个栈的栈顶。在一开始的时候，$p_1$指向数组的0号位置，$p_2$指向数组的n+1号位置。进行push操作时，一个栈向左扩充一个栈向右扩充，当两栈的栈顶相遇时则溢出。进行pop操作时，两栈分别向两边缩减直到回到初始位置。

  • 这种题画个图结合一下数据结构特性要解应当不难，栈的特性就是后进先出，可以联想一堆叠在一起的椅子。

P6 用两个先进先出队列实现一个栈

  • 主要思路：

    设置P、Q为两个先进先出队列，进行push操作就是将新元素加入到P队列中。进行pop操作就是不断将P中的元素拿出来放到Q中直到P中只剩一个元素。删除P中的最后一个元素（根据FIFO和栈各自的特性该元素即为实现的栈的栈顶）。最后将Q中的元素拿出来再全部放到P中去。

  • 队列的操作有：Enqueue和Dequeue

P7 随机队列
  • 题意：实现一个特殊的队列，remove操作是随机选取元素删除。题目看着唬人，其实水题。
  • 遇到这种看上去创新的题目，把题目好好读下来就好啦。记得看有没有正确性证明的要求什么的。
